DEPENDENCIES:

You might need to install some of these before running the game:

  Python:     		 http://www.python.org/
  PyGame:    		 http://www.pygame.org/
  PyOpenGL:   		 http://pyopengl.sf.net/
  
  Psyco(Optional):	 http://psyco.sourceforge.net/

 
RUNNING THE GAME:

On Windows or Mac OS X, locate the "run_game.pyw" file and double-click it.

Othewise open a terminal / console and "cd" to the game directory and run:

  python run_game.py



HOW TO PLAY THE GAME:

The controls are:
    Up      = W
    Left    = A
    Down    = S
    Right   = D
    Jump   = Left Control
    Fire    = Left Alt

To Enter or Exit Full Screen Mode use the key combinatio Alt + Enter at the menu screen.
After leaving the menu, the game will be locked into full screen or windowed.

To quit, simply hit the escape key.
